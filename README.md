<details open>

<br>

Frontend Engineer on a 4-person cross-functional team, building a fraud-detection prototype that flags mule bank accounts. I own the frontend architecture and integrate fraud-risk signals from the ML/graph-anomaly pipeline into an interface analysts can actually use.

`React` `TypeScript` `Vite` `Tailwind CSS`

</details>

<details>
<summary><b>⛓️ BLAST — Blockchain Ledger & Audit System</b> &nbsp;<code>Blockchain Core Developer</code></summary>
<br>

A tamper-proof banking transaction ledger on a private Ethereum blockchain (Ganache). SHA-256 block hashing keeps the chain honest; Web3j syncs transactions on-chain; JWT auth and three-tier RBAC (Admin, Auditor, User) lock down access.

`Java 17` `Spring Boot 3.x` `Spring Security` `MySQL` `Web3j` `Maven`

</details>

`HTML` `CSS` `JavaScript`
